Basic example:

```tsx
<Form>
    <input
        type="text"
        style={{ border: '1px solid black', margin: '1em', padding: '1em' }}
        placeholder="type something"
    />

    <input
        type="text"
        style={{ border: '1px solid black', margin: '1em', padding: '1em' }}
        placeholder="type something else"
    />

    <div>
        <button
            type="text"
            style={{
                border: '1px solid black',
                margin: '1em',
                padding: '1em',
                cursor: 'pointer',
            }}
            placeholder="type something else"
        >
            Ok
        </button>
    </div>
</Form>
```
